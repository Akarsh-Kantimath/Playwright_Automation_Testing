In order to read, write and modify the excel using Javascript
node module - excel JS which will help us to read, manipulate and write into excel and json files.
https://www.npmjs.com/package/exceljs

1. create a node project
2. go to gitBash and create a folder mkdir ExcelJSUtil
3. cd ExcelJSUtil
4. npm init 
5. hit Enter
6. npm install exceljs -> It will download all the Plugins related to excelJS
7. now the node_modules folder is created.
8. Instead of step 6 command if we use
npm install exceljs --savedev -> entry of excelJS pulgin is recorded into package.json file
//If the entry of dependencies present in package.json
//Once we send zip.file of package json user can install all dependencies of project by command -> npm install -> users need not look for excelJS 
plugin

	"exceljs" : "4.4.0"
}

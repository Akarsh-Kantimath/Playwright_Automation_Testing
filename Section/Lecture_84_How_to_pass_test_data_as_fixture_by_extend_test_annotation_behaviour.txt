How to pass test data as a fixture by extend test annotation behaviour
----------------------------------------------------------------------
We can pass the testdata as a fixture - Limitations is to pass only one single set of test data.

We are creating a custom fixture to extend the test data set to pass through test fixture

1. Inside utils folder create test-base.js file
2. test-base.js

const base = require('@playwright/test');

// Extend base test by providing testDataForOrder option.
const customtest = base.test.extend({
  testDataForOrder: {
    username: 'tester_akarsh@gmail.com',
    password: 'Abcd1234#',
    productName: 'iphone 13 pro',
  },
});

// Export the extended test so other files can use it.
module.exports = { test: customtest, expect: base.expect };

3. Now, import the test data file by the folder location

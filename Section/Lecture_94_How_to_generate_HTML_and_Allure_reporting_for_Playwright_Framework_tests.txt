How to generate Allure reporting
=================================
It's a 3rd party plugin

google allure playwright to get node package details.
https://www.npmjs.com/package/allure-playwright

install node package manager through
for windows: npm install -g allure-commandline --save-dev
npm install -D allure-playwright

Note: before writing the before command Java Development Kit JDK to be installed

npx playwright test --reporter=line,allure-playwright

through =line ->It's playwright test report how many pass and fail that act as input to allure-reporting

1. Once after test execution is completed, allure-results folder is created.
2. In the terminal, type the command
allure generate ./allure-results --clean 
//It's like asking allure to generate report by going through ./allure-results folder and to clean up if their exists any previous test results.

3. Upon successful generating the report, another folder is created with name allure-report
By using the command
allure open ./allure-report

=================================================================
In the playwright.config.js file

reporter : [['html'],['line'],['allure-playwright']],

to be added

// @ts-check
const { defineConfig, devices } = require('@playwright/test');

const config = ({
  testDir : './tests',
  retries : 1 , // Maximum time one test can run for
  workers : 1, //Mentioning number of worker for test execution of files parallel
  timeout : 15 * 1000,
  expect : {
      timeout : 5 * 1000,
  },
  reporter : [['html'],['line'],['allure-playwright']],
  use : {
      browserName: 'chromium',
      headless: true,
      screenshot: 'on', //'off', 'on', 'only-on-failure'  //'on' - captures screenshot after each test.
      trace: 'retain-on-failure' //'on', 'off' , 'retain-on-failure'
  },
});

module.exports = config
